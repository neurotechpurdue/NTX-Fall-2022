# BASE FOR PURDUE NTX FALL 2022 PROJECT
Project start date: September 24th, 2022
Project sumbission date: November 1st, 2022

# GOALS
The goal of this project is to create an interface to assist the user in controlling HTML5
videos without the need to use.
We will be using motor imagery to accomplish this goal, and we are aiming for an accuracy of 90%
to increase normalcy in the lives of those who struggle to perform these tasks.
